lists in html

HTML lists allow web developers to group a set of related items in lists. An ordered list starts with the <ol> tag. Each list item starts with the <li> tag
    //A,a,i
  1. ABC
  2. DEF
  3. GHI
An unordered list starts with the <ul> tag. Each list item starts with the li tag. The list items will be marked with bullets (small black circles) by default
  • ABC
  • DEF
  • GhI
